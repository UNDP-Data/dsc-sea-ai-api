This is a beta release of the API that includes a streamlined data schema and improved performance.


What's New
  • /graph endpoint returns an object containing nodes and edges objects to make data visualisation straightforward.
  • /model endpoint now support a list of messages (conversation history ) and returns model response together with relevant documents and a ready-to-use graph.
  • LanceDB is now used for storing the data as well as full-text and vector searches.
  • Overall performance has been improved due to GPT-4o mini and LanceDB.

Bug Fixes
  • None.
