This release refines the knowledge graph and graph query process.


What's New
  • The knowledge graph now includes over 20k nodes and 50k edges extracted from SDG 7-related UNDP publications and SDG 7 reports.
  • node's metadata and edge's level properties have been removed.
  • Graph data is traversed using an in-memory NetworkX graph.
  • An ETL nodebook for updating database records has been added.
Bug Fixes
  • Fixed broken tests.
