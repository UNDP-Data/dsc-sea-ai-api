This release includes new endpoints for nodes, support for streaming model responses and key-based authentication.


What's New
  • BREAKING CHANGE all endpoints now require authentication with X-Api-Key in the header.
  • BREAKING CHANGE response from /model> endpoint is now streamed in NDJSON format. Each object is AssistantResponse as before but only the first one contains ideas, documents and graph. content property only includes new content produced by the model (delta streaming).
  • Two new /nodes endpoints have been added to list all available nodes and retrieve data for a single node.

Bug Fixes
None.
