This release enhances RAG performance of the /model endpoint and includes an important bugfix.


What's New
  • The RAG flow now uses a new collection of SDG 7 reports and UNDP documents related to the energy topic.
  • Embedding engine has changed to text-embedding-3-large, using reduced vector space (1024 dimensions).
  • The underlying chat model has been upgraded to gpt-4.1-nano.
  • RAG performance and quality have been improved by tuning the prompts and increasing the limit of documents for retrieval. A custom reranker is now used to promote results from more recent documents.
  • The use of SQL tools is experimental and not encouraged in the model instructions.

Bug Fixes
  • Previously, document content was not correctly passed to the model, making RAG rely on document summaries only. This has been fixed now.
